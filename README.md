## Project Overview
Full 3-statement financial model and DCF 
valuation for Tesla Inc. (TSLA) with automated 
daily monitoring system.

## What's Included
- Income Statement (2021-2028E)
- Balance Sheet
- Cash Flow Statement
- DCF Valuation with WACC
- Sensitivity Analysis (WACC vs Terminal Growth)
- Scenario Analysis (Bear/Base/Bull)
- Automated daily price monitoring via n8n

## Key Findings
- Bear Case: $20
- Base Case: $25
- Bull Case: $35
- Current WACC: 12.4%
